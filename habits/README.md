# Habits (Vue demo)

A daily habit tracker built with Rails, Inertia.js, and Vue 3. Pure Normal Mode, no JavaScript bridge code required.

## Features

- Create habits with custom colors
- Track daily completions
- Today view with quick toggles
- Native tab bar, nav bar, push notifications
- Dark mode

## Quick start

```bash
npm install
bin/setup
bin/dev
```

Runs on port 3004. The dev server starts Rails and Vite.

Default login: `user@example.com` / `password`
